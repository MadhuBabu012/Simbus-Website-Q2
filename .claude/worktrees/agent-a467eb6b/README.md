# Simbus Technologies Website

Static marketing website for Simbus Technologies — a Kinaxis Maestro + Databricks specialist consulting firm.

## Stack

Pure HTML / CSS / JS. No frameworks.

## Structure

```
html/          # 8 pages
css/styles.css # shared stylesheet
js/main.js     # dark mode, mobile nav, scroll reveal, counters
assets/        # images, video
```

## Pages

`index` · `about` · `services` · `kinaxis` · `databricks` · `careers` · `contact` · `roi-calculator`

## Run locally

```bash
python3 -m http.server 8080
# http://localhost:8080/html/index.html
```
